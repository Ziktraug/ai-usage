# Security policy

Please report suspected vulnerabilities privately through [GitHub Security Advisories](https://github.com/Ziktraug/ai-usage/security/advisories/new).

Do not include real prompts, local history files, usage databases, credentials, tokens, or private configuration. Use a minimal synthetic reproduction and describe the affected version or commit, impact, and reproduction steps.

Public issues are appropriate for ordinary bugs that do not expose private data, bypass a trust boundary, execute unintended local operations, or make a non-loopback service reachable.

This project does not promise a response or remediation SLA. Supported behavior and trust boundaries are documented in [docs/architecture.md](docs/architecture.md); older revisions may not receive security fixes.
