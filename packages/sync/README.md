# @ai-usage/sync

Retired legacy snapshot sync package.

This package is kept only for the existing CLI snapshot-sync command surface while LAN merge replaces UI/reporting sync behavior. New LAN pairing, peer discovery, authenticated merge bundle exchange, and peer imports live in `@ai-usage/lan-pairing`, `@ai-usage/usage-merge`, and `@ai-usage/usage-store`.

Do not add new web or report-data dependencies on this package.
